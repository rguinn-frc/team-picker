NHL 26 Team Randomizer

This tool will create a balanced three‑game round‑robin tournament for NHL 26. Use the same seed to share results – by default the seed is based on the current hour in Denver (America/Denver), so everyone will see the same matchups. At the end of your three games you can crown the GCD (Greatest Common Denominator) if someone wins all games or the LCD (Lowest Common Denominator) if someone loses them all.

Note: Overpowered teams (Tampa Bay Lightning and Colorado Avalanche) are excluded from selections. The higher‑rated team in each matchup is highlighted for Nick. Select a winner for each matchup and use the button above to copy your results and open the shared tracker.
